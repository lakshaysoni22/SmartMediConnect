## How to Setup (5 minutes)

### Step 1: Get Free Web3Forms API Key
1. Visit https://web3forms.com
2. Click "Get Started Free"
3. Enter your email: **lakshaysoni012794@gmail.com**
4. Verify your email
5. Copy your **Access Key** (looks like: `8e5c7c9a-5e8a-4c8a-9e0a-1c8e7a5c9a8e`)

### Step 2: Update the Code
1. Open `/components/LogoutFeedbackModal.tsx`
2. Find line ~35:
   ```typescript
   access_key: '8e5c7c9a-5e8a-4c8a-9e0a-1c8e7a5c9a8e', // ← Replace this
   ```
3. Replace with your real Access Key:
   ```typescript
   access_key: 'YOUR_ACTUAL_ACCESS_KEY_HERE',
   ```
4. Save the file

### Step 3: Test It
1. Login to any portal (Patient/Doctor/Admin)
2. Click Logout
3. Fill the feedback form
4. Click "Submit & Logout"
5. Check **lakshaysoni012794@gmail.com** inbox!

## Troubleshooting

### Emails not arriving?
1. Check spam folder
2. Verify Access Key is correct
3. Check browser console for errors (F12)
4. Ensure internet connection is active

### Want to change recipient email?
1. Open `/components/LogoutFeedbackModal.tsx`
2. Find line ~38:
   ```typescript
   email: 'lakshaysoni012794@gmail.com',
   ```
3. Replace with new email address
4. Update Web3Forms account with same email
